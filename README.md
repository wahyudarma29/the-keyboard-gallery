# Keyboard Gallery
If you want to run the project locally, you can follow these steps:

1. Clone the repo: 
`git clone https://github.com/wahyudarma29/the-keyboard-gallery.git`

2. Install dependencies: 
`npm install`

4. Run prisma generate: 
`npx prisma generate`

5. Migrate the db schema: 
`npx prisma migrate dev`

6. Run the seed: 
`npm run seed`

After that you can run it locally using `npm run dev`

------------


As for now, the web has features as follows:
1. Can add new user
2. Auth
3. User can add their own gallery
4. User can favorites other people gallery
5. User can also see what gallery that has been liked
